# The four views — what each screen actually shows (BD-approved 2026-07-21)

The player is ONE agent (the organizer) in a population of agents — the state,
corporations, institutions, mass orgs are CPU agents driven by deterministic policies.
Every view is a read-only projection of the world graph; the verbs are the only writes.

## 1. Dashboard — the economy dossier

- **The Fundamental Theorem verdict, front and center**: Wc vs Vc (core wages vs value
  produced). While Wc > Vc, revolution in the core is impossible — show the gap as the
  headline gauge with the crimson/gold ramp.
- **Imperial Rent Φ tri-decomposition**: φ_unequal-exchange + φ_reproduction +
  φ_domestic — stacked bars or a three-column plate.
- Surplus split s = p + i + r + t (profit, interest, rent, tax) as a partition bar.
- Survival calculus: P(S|A) sigmoid vs P(S|R) = Organization/Repression — rupture
  fires when P(S|R) > P(S|A); show both needles.
- Matter book strip: overshoot O = C/B, the ratcheted ceiling M̄ (only ever falls),
  and an HONEST ABSENCE cell for the energy vertex ("β_J — UNPOSITIONED") — absence
  is rendered, never faked.
- All aggregates are ratio-of-sums; per-class rows, never averaged across classes.

## 2. Map — the geographic instrument

- County choropleth (3,153 counties nationwide; Wayne County, MI is the tutorial).
- **Lens bar**: value / tension / wealth / wage-pressure / fog lenses; a β-simplex
  lens (Land–Labor–Energy composition) arrives later.
- Muted cartographic fills (the KSBC palette stays for chrome; the map keeps
  discipline) with crimson/gold markers for ruptures and player assets.
- Fog of war is epistemic: unknown counties render as static/hatch, not as zeros.

## 3. Wiki — the Archive itself

- The game IS a wiki: every entity (class, org, county, figure, event) is a page in
  a versioned vault; events revise pages; the player reads history as it's written.
- Wikilinks everywhere; backlink index panel; redlinks for known-unknowns.
- Narrative prose blocks (the AI narrator) render in fenced wells with a quill mark —
  clearly typed as observation, never as engine truth. Degraded narrator renders an
  absence fence, honestly.

## 4. Topology — the relational instrument

- Orgs, institutions, coalitions, the state, mass orgs as a text-first incidence/
  Levi graph (ASCII floor), with an optional raster panel (kitty graphics lane).
- Edges are typed: EXPLOITATION, SOLIDARITY, WAGES, TRIBUTE, TENANCY, ADJACENCY.
- Individuals and coalitions are NOT in the world yet — render them as declared
  future absence (dim "no such stratum yet" wells), never as fake nodes.

## Constant chrome

- Chronicle rail (left): tick-stamped events, severity-tiered, CRITICAL autopauses.
- Watchlist rail (right): pinned entities, tick-fresh gauges.
- Verb bar (bottom): 9 verb keys, cost chips, target pickers; disabled verbs show WHY.
- Endgame strip: five axes (REVOLUTIONARY_VICTORY, ECOLOGICAL_COLLAPSE,
  FASCIST_CONSOLIDATION, RED_OGV, FRAGMENTED_COLLAPSE) as continuous mini-gauges.
