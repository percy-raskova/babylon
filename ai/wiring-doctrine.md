## 0. Why a doctrine

Every disconnection we keep finding has the same anatomy: a producer whose write-set
nothing reads, a consumer whose read-set nothing writes, a gate no production supplier
satisfies, or a stub feeding a live consumer. In the formalism these are all the same
object — a nonempty boundary ∂L between the declared construct graph and the live
subgraph. The T1.1 seam-algebra sentinel now *detects* that boundary; this doctrine says
how to *close* it without ad-hoc calls that rot into the next generation of gaps.

## 1. The classification key (decidable from the diff — VII)

Ask three questions, in order:

1. **Does it need a new sort, node/edge kind, or constructor family?** → Modulus 𝕊/𝔇:
   amendment territory (the `Matter` sort, the energy accounts → Amendment AB). Nothing
   else in the wire waits for the amendment — split the C/G/P remainder out and land it.
2. **Is it a new term over existing generators** (a system, an opposition + measure, a
   coupling row, a projection/page kind, a verb target-sort)? → Modulus C/G/P: normal
   development, NO amendment. Obligations: A1 manifest ε-row, III.10 rent tag, one
   behavioral artifact per new boundary, scoped tests.
3. **Is it a coefficient?** → Modulus Θ: lands in GameDefines/A6 registry with a declared
   tier (Θ_data / Θ_theory / Θ_feel) and envelope. A magic literal in a module dict
   (`_CLASS_INHERITANCE_SCALE`, the 0.6/0.4 splits) is a Θ-shaped quantity squatting
   outside the registry — rehome it before wiring anything through it.

## 2. The five canonical wiring motions

**W-C — the dataflow wire.** A wire between two systems IS a declared nonempty
intersection `W(producer) ∩ R(consumer)` on a named field — a `(node_sort, attribute)`
pair, an edge sort, a graph register, or an event topic. Enter it as manifest ε-rows on
both sides; it becomes an edge in the conflict DAG (T-4) and the materialist-causality
audit checks its direction (base never reads what superstructure wrote this tick).
Never wire by importing and calling — wire by writing a field the consumer already
declares it reads, or declaring the read. Gated dormancy (`context.get(K)` no one
supplies, optional kwargs never passed) closes by giving the gate a production supplier
and a gate-satisfaction sentinel row. *Applies to: doctrine→bifurcation feedback (DT
Unit 6), vol2_step lighting, LODES hydration kwargs, legitimation_state writeback.*

**W-𝔇 — the opposition registration.** Any state-facing scalar tension becomes a bound
opposition: `OppositionSpec` + measure (pure function of shipped fields, fresh per tick,
NO accumulator), UNPOSITIONED honest absence when inputs vanish, `shadow=True` first per
the ADR077 promotion ladder, coupling rows into the CouplingGraph (the five ratified
kinds). Principal-contradiction selection, ⊗/⊕ composition, and the I.13 scoring inherit
for free — rows, never rules. *Applies to: `metabolic` (ships against existing kernels
today), `somatic` (waits on J), the σ-gradient's cut-tension (spec-107).*

**W-G — the scale wire.** Cross-resolution data binds through an adjunction
(ScaleAdjunction: allocate ⊣ aggregate; Aufhebung between levels), never by stamping the
finer sort onto the engine graph. A finer-grain store with no adjoint to the grain the
engine mutates is dead schema (the hex-biocapacity estate; the empty-set county
aggregation). *Applies to: hex biocapacity → county territories, LODES OD matrices,
the county-B construction reconciled to the national FoDaFo series (a conservation law,
not a vibe).*

**W-P — the projection wire.** Observation surfaces are one-way functors: DeclaredView
+ fence + fog gate; recognizer inputs allowed, control inputs FORBIDDEN (Amendment-S
tripwire — the paper's own cut of replicator dynamics on the β simplex is binding
precedent). Player-relative knowledge lives here (fog = epistemic interior; never in
the tick hash). *Applies to: EpistemicHorizon Phase 2 (its outputs currently feed only
the legacy web bridge), the β chart, v_*_trend views, the economy dossier, derived
severity.*

**W-A4 — the conservation closure.** Every new flow declares a budget row: Src set,
Snk set, residual evaluator, residual = ALARM (III.11), never a warning (the F-3
pattern is the anti-pattern). A live consumer fed a literal/neutral constant while a
registered calculator exists is a stub-liveness violation (the
`ReproductionBalance(condition_met=True)` class) — the stub-vs-calculator sentinel
reds it. *Applies to: φ_repro = Σ shadow subsidy (checked against L-VAL-5), the matter
budget L-MAT-1, population conservation (F-3 warn→alarm), the carbon register.*

## 3. The surfaces: verbs and events

- **Verbs**: mechanics reach the player/CPU surface ONLY as products — verb ×
  target-sort × parameter — on the fixed 9 player + 6 state generators. L-ATOM (one
  graph op, one target), L-DOM (Investigate enlarges the domain), L-PATH (edge-mode
  ladder, no shortcuts), L-SPEND (verbs spend, never mint — no resolver carries
  `creates_value`). Propagated consequences land on the FIXED instrument panel
  (opposition gauges, P(S|A)/P(S|R), pool, heat) — new *pathways*, never new needles.
  A tenth verb is a constitutional event and should feel like one.
- **Events**: new wiring emits catalog kinds (ALARM / CROSSING / FLOW / ACT / PATTERN)
  with severity DERIVED from kind × terminal proximity (R-EC-1) — never a hand-tiered
  dict, never routed through `logging` (the Observability Spine subscribes one-way).

## 6. Sentinel closure (the standing rule applied)

Each motion class has its gate: W-C → gate-satisfaction + disconnected-subsystem (∂L,
shipped in T1.1); W-𝔇 → catalog property tests + vocabulary; W-G → the wrong-rung
territory-keying rule + empty-iteration detection (NEW — charter it: a fold over a node
sort no production path stamps is the hex-aggregation bug class); W-P → import-linter
one-way lattice + Amendment-S grep-gate (widen to `formulas/`, owed from T1.1); W-A4 →
conservation registry + stub-vs-calculator. A wiring PR that adds no sentinel row for
its class is incomplete by definition.

## 7. Enforcement charter (ratified with the doctrine — ADR109)

**Effective immediately, no code required:** a wiring PR that does not ship its motion
class's sentinel row — or cite the dependency that blocks it — is **incomplete and
review-rejectable**. This binds every post-cascade lane from ratification day.

now: it EXTENDS estates still sitting unmerged on lane branches — T1.1's
